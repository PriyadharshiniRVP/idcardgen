# ID Card Generator with Student Management System

## About
A web-based application for managing student records and generating ID cards. The system allows administrators to perform CRUD operations on student data and maintain records in a PostgreSQL database.

## Features
- Add new students
- View student records
- Update student information
- Delete student records
- Generate ID cards
- Database integration using PostgreSQL
